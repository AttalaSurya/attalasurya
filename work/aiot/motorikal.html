Nusantara

Musikal Nusantara merupakan permainan musikal dengan tema lagu daerah. Terada pat fitur lirik sehingga anak-anak dapat bernyanyi bersama ketika bermain. Permainan ini dapat digerakkan dengan kursor, sentuhan layar, ataupun dengan gerakan tangan menggunakan alat (Hardware) Motorikal. Permainan ini dapat dimainkan melalui platform windows ataupun android. Download

Motorikal

Motorikal merupakan alat yang digunakan untuk menangkap gerakan tangan lalu dikirimkan pada game sebagai gerakan. Alat ini dapat bekerja dengan kabel ataupun tanpa kabel. Pengiriman data dilakukan dengan usb atau bisa menggunakan bluetooth. Dapat digunakan untuk windows ataupun android.
